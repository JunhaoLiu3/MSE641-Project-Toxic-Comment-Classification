1. 平时不跑 BERT - 适合你只想跑 Part 1–3 / Part 5： FF

python - <<'PY'
from pathlib import Path

p = Path("final_project.py")
s = p.read_text()

s = s.replace("RUN_BERT = True", "RUN_BERT = False")
s = s.replace("BERT_DEBUG = True", "BERT_DEBUG = False")

p.write_text(s)
print("Set to: RUN_BERT=False, BERT_DEBUG=False")
PY

确认：
grep -n "RUN_BERT\|BERT_DEBUG" final_project.py
